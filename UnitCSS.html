หน่วย(Unit)ใน CSS(Absolute)แบบตายตัว

Unit Detail
px pixel(พิกเซล)เป็นหน่วยที่นิยมใช้มากที่สุด โดยที่ 1px = 0.75pt สัมพันธ์กับรายละเอียดหน้าจอ
pt point โดยที่ 1pt = 1/72 inchs ใช้ในงานสิ่งพิมพ์
cm เซนติเมตร
mm มิลลิเมตร
in inch (1 in = 96px =2.54cm)
pc picas(1pc = 12pt)

หน่วย(Unit)ใน CSS(Relative)แบบอัตราส่วน

% Detail
em
    อ้างอิงขนาดกับ element parent ที่ใกล้ที่สุด ใช้ในการกำหนดขนาดจำนวนเท่าของขนาดปัจจุบันเช่น หากขนาดที่ใช้ตอนนี้คือ10px
  • ถ้ากำหนดนาดเป็น 2em จะหมายถึง 2 เท่าของขนาดปัจจุบัน คือ 20px
  • ถ้ากำหนดนาดเป็น 1.4em จะหมายถึง 1.4 เท่าของขนาดปัจจุบัน คือ 14px
  • ถ้ากำหนดขนาดเดิมให้กำหนดเป็น 1em = 10px
rem กำหนดขนาดโดยอ้างอิงกับ root element ปกติ font-size จะอยู่ที่16px
vw
    1% หรือ 1/100 ของviewport height
  • Width ของbrowser viewport เท่ากับ 750px ค่า 1vw = 7.5px
vh height ของ browser viewport เท่ากับ 900px ค่า1vh = 9px
vmin,vmax กำหนดค่าต่ำสุด และสูงสุดของviewport
